# ReelCraft AI

AI-powered reel generation platform built using Flask, FFmpeg and gTTS.

## Features

* Upload multiple images
* Generate AI narration using gTTS
* Upload custom audio files
* Custom reel naming
* Adjustable image duration
* Caption overlays
* Download generated reels
* Delete generated reels
* Image preview before generation

## Tech Stack

* Python
* Flask
* FFmpeg
* gTTS
* HTML
* CSS
* JavaScript

## Installation

```bash
pip install -r requirements.txt
python main.py
```
